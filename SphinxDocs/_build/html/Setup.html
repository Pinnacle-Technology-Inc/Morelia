class Setup.Setup_PodDevices.Setup_PodDevices(saveFileDict: dict[str, str] | None = None, podParametersDict: dict[str, dict[int, PodApi.Parameters.ParamsBasic.Params] | None] | None = None)

Bases: object

Setup_PodDevices allows a user to set up and stream from any number of POD devices. The streamed data is saved to a file.

REQUIRES FIRMWARE 1.0.2 OR HIGHER.
